## How to Push Code to GitHub

To save your work to your GitHub repository at `https://github.com/amritanshushivam/Pirona_moments`, follow these steps.

### One-Time Setup

First, you need to connect this project to your GitHub account. Open your terminal and run these two commands **only once**:

```bash
git remote add origin https://github.com/amritanshushivam/Pirona_moments.git
git branch -M main
```

### Pushing Future Changes

Anytime you want to save your changes to GitHub, just run this single command in your terminal:

```bash
sh ./push-to-github.sh
```


This script will automatically add, commit, and push all your code for you.
