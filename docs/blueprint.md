# **App Name**: VivahVerse

## Core Features:

- Vendor Directory: Browse and filter vendors by service, location, budget, and ratings.
- AR Decor Preview: Visualize decorations in your venue using augmented reality.
- AI-Powered Package Builder: Automatically suggest a tailored set of services and vendors to create an event package that suits user preferences. LLM as tool.
- AI Fraud Detection: Analyze vendor data to detect potentially fraudulent listings and alert users.
- Secure Payment System: Integrate with payment gateways (UPI, Razorpay, Paytm) for secure transactions and EMI options.
- Customer Dashboard: Manage bookings, payments, saved vendors, and track budget.
- Vendor Dashboard: Manage profile, services, bookings, and track earnings and use fraud detection alerts.

## Style Guidelines:

- Primary color: Vibrant Coral (#FF7F50) to represent energy and festivity.
- Background color: Soft Peach (#FAE0D4) to create a warm and inviting atmosphere, complementing the coral without overpowering it.
- Accent color: Golden Yellow (#FFC857) to highlight important elements such as CTAs, enhancing the user experience.
- Headline font: 'Belleza' sans-serif, lending personality to the design; body text: 'Alegreya' serif, chosen to ensure excellent readability for long content.
- Use culturally relevant and celebratory icons.
- Modern and clean layout with a focus on visual hierarchy.
- Subtle and elegant animations for page transitions.
